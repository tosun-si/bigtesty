# Contributing

## Prerequisites

- [python](https://www.python.org/)
- [pre-commit](https://pre-commit.com/)
- [pipenv](https://pipenv.pypa.io/en/latest/)

## How to run

```bash
git clone https://github.com/tosun-si/bigtesty
pipenv shell
pip install -e .
```

## Docker build

```bash
docker build -t mazlumtosum/bigtesty .
```
